Aprendendo as primeiras tags

Na tag < title > < /title >, o texto acrescentado aparece na guia da internet.

Para o título aparecer no corpo do site deve-se usar a tag < h1 > < /h1>

A tag < hr> acresenta uma linha horizontal.

A tag < br > quebra a linha em um lugar específico. Ex.:

SEM A TAG: Lorem ipsum dolor sit amet consectetur adipisicing elit. Eligendi adipisci, exercitationem eos tenetur, reiciendis explicabo asperiores atque facilis ab eaque natus iure doloremque saepe fugiat nisi, porro voluptates illum id.

COM A TAG: Lorem ipsum dolor sit amet consectetur adipisicing elit. Eligendi adipisci, exercitationem eos tenetur, reiciendis < br >
explicabo asperiores atque facilis ab eaque natus iure doloremque saepe fugiat nisi, porro voluptates illum id.

Acrecentar a tag < !---- > adiciona comentários, que não aparecerão na página da internet.< !--Este é um exemplo de comentário-- >.

Adicionando imagens

Usar < img src="" alt="" > Com o cursor dentro das aspas de "src" pressionar CTRL+ESPAÇO, aparecerá a lista de imagens que estão dentro da pasta de trabalho html.
alt="" é o texto alternativo, uma explicação da imagem que está sendo utilizada.

Exemplo: < img src="imagens/paisagem-650.jpeg" alt="paisagem com árvore em primeiro plano, um lago e uma montanha ao fundo">

Imagens externas: copiar e colar o link de uma imagem da internet.

Exemplo:< img src="https://i0.wp.com/emotioncard.com.br/wp-content/uploads/2016/03/10-nat1.jpg?w=800&ssl=1" alt="paisagem pequena cachoeira">

Marca-Texto

Para criar um texto marcado utiliza-se a tag < mark > < /mark >.

Texto grande e pequeno

Para destacar um trecho com letras pequenas utiliza-se a tag < small > < /small > .

Texto deletado e texto incluído

Para marcar um texto como excluído, indicando que ele deve ser lido mas não considerado, utiliza-se a tag < del > < /del>.

Para marcar um texto como inserido, enfatizando que ele foi adicionado depois utiliza-se a tag < ins > < /ins>.

Texto sobrescrito e subscrito

Para texto sobrescrito utiiza-se atag < sup > < /sup>. Exemplo:x< sup >20 < /sup > +3 retorna: x20+3

Para texto subscrito utiliza-se a tag < sub > < /sub>. Exemplo:H < sub> 2 < /sub > 0 retorna: H20

Inserindo código-fonte em páginas

A tag < code > é usada para mostrar um trecho de código fonte, como uma instrução ou uma função. Ela preserva a formatação do código, mas não o executa.

Exemplo: document.getElementById('teste')

Para manter a formatação de códigos grandes utiliza-se o, < code > dentro da tag < pre > < /pre>.

Exemplo:
< pre >
... < code >
1 x +[1,2,3]
2 y = x
3 x.append(4)
4 print(y)
...< /code >
< /pre >

Citações

Para citações sem quebra de linha utiliza-se a tag < q > < /q >.

Para uma citação em bloco, com quebra de linha, utilza-se a tag < blockquote > < /blockquote >. A citação aparecerá deslocada. É possível criar o parâmetro: cite= e inserir o link da citação. Este parâmetro é importante para os mecanismos de busca.

Abreviações

Utiliza-se a tag < abbr title=""> < /abbr >. Em title= digite o significado da sigla ou abreviação. O resultado na página será que ao passar o mouse por cima aparecerá o conteúdo do title

Texto invertido

Para inverter um texto utiliza-se a tag < bdo dir=""> < /bdo >. Em dir= digite rtl para o texto ser exibido da direita para esquerda. Digite ltr para exibir o texto da esquerda para a direita.

Listas

Listas Ordenadas (OL)

Utiliza-se a tag < ol > < /ol > e dentro dela a tag < li> < /li>, que são os itens de lista. O < /li > é opcional.

< ol>
< li>Acordar< /li>
< li>Tomar café< /li>
< li>Trabalhar< /li>
< /ol>

O parâmetro type=, dentro de < ol>, permite alterar o caractere da numeração: pode ser 1, A, a, I ou i.

Se quiser que a lista seja numerada apartir de úm número ou letra diferente, adicione depois do type= o parâmetro start= e digite o número ou letra.

Listas não ordenadas (UL)

Utiliza-se a tag < ul> < /ul> e dentro dela a tag < li> < /li>. Nestas listas a ordem dos itens não é importante. Para alterar o caractere dos marcadores utiliza-se o parâmetro type=. Os tipos podem ser: circle, disc ou square.
